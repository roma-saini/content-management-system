# Content Management System (CMS)

A Content Management System (CMS) built with PHP and MySQL that enables administrators to manage website content through an intuitive admin dashboard. The application supports user authentication, category management, content publishing, image uploads, and complete CRUD operations.

## Features

- User authentication (Admin Login)
- Admin dashboard
- Create, Read, Update, and Delete (CRUD) operations
- Category management
- Content management
- Image upload support
- Responsive interface
- MySQL database integration

## Technologies Used

- PHP
- MySQL
- HTML5
- CSS3
- JavaScript

## Project Structure

```
content-management-system/
│── admin/
│── includes/
│── uploads/
│   └── images/
│── index.php
```

## Installation

1. Clone the repository.

```bash
git clone https://github.com/roma-saini/content-management-system.git
```

2. Move the project to your XAMPP `htdocs` folder.

3. Create a MySQL database and import the required SQL file.

4. Update the database configuration in the project if required.

5. Start Apache and MySQL using XAMPP.

6. Open your browser and visit:

```
http://localhost/content-management-system/
```
