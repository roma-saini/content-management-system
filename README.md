# Content Management System (CMS)

A Content Management System (CMS) developed using PHP and MySQL that enables administrators to create, manage, edit, and publish website content through a secure admin dashboard.

## Features

- User Registration & Login
- Secure Authentication
- Admin Dashboard
- Create, Read, Update, and Delete (CRUD) Content
- Category Management
- Content Publishing
- Search Functionality
- Responsive User Interface
- MySQL Database Integration

## Technologies Used

- PHP
- MySQL
- HTML5
- CSS3
- JavaScript

## Installation

1. Clone the repository.

```bash
git clone https://github.com/roma-saini/content-management-system.git
```

2. Move the project to the XAMPP `htdocs` folder.

3. Import the MySQL database into phpMyAdmin.

4. Start Apache and MySQL.

5. Open:

```
http://localhost/content-management-system/
```
